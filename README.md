# 🍄 MS Canopy — Premium Exotic Mushrooms Website

A professional business website for **MS Canopy**, built with **React + Vite**.

---

## 🚀 Quick Start (Local Development)

### 1. Install dependencies
```bash
npm install
```

### 2. Run the application
```bash
npm run dev
```
- Frontend → http://localhost:3000

---

## ☁️ Deployment (GitHub Pages)

This project is configured to deploy automatically to GitHub Pages via GitHub Actions.

1. **Push to GitHub**: Every push to the `main` branch triggers a build and deploy.
2. **Settings**: On GitHub, go to **Settings** -> **Pages**.
3. **Build and deployment**: Ensure "Source" is set to "GitHub Actions".
